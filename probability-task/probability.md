# Probability Task

## Problem

A coin is tossed **three times**.  
What is the probability that **exactly two heads** will appear?

---

## Solution

Each coin toss has **two possible outcomes**:

- H — Head
- T — Tail

Since the coin is tossed **three times**, the total number of possible outcomes is:

2³ = 8

### All Possible Outcomes

| Outcome |
|--------|
| HHH |
| HHT |
| HTH |
| HTT |
| THH |
| THT |
| TTH |
| TTT |

Total outcomes = **8**

---

### Outcomes with Exactly Two Heads

| Outcome |
|--------|
| HHT |
| HTH |
| THH |

Number of favorable outcomes = **3**

---

## Probability Formula

Probability = (Number of favorable outcomes) / (Total number of outcomes)

P = 3 / 8

---

## Final Answer

P = **0.375**

or

**37.5%**
